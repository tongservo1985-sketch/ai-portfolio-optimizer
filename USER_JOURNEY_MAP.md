# User Journey: The 3-Minute Bespoke Pitch

| Phase | User Action | System AI Action | Emotional State |
| :--- | :--- | :--- | :--- |
| **1. Trigger** | Finds an interesting job on LinkedIn/Upwork. | N/A | Excited but Anxious |
| **2. Input** | Pastes the job URL into the Overlord Assistant. | Parses the text, identifies that the client needs "Modern SaaS Branding." | Hopeful |
| **3. Matching** | Clicks "Match My Work." | Searches 500+ past assets; selects 4 branding projects and 2 UI kits that specifically use a similar color palette/industry. | Impressed |
| **4. Refinement** | Swaps out one project for a more recent one. Edits the AI-suggested intro. | Updates the layout instantly. Regenerates the "Why this matters" snippet for the new project. | In Control |
| **5. Delivery** | Hits "Publish & Copy Link." | Generates a high-performance web page and sets up tracking pixels. | Professional |
| **6. Result** | Sends link to client. | Monitors visitor behavior. Sends push notification: "Client in London is viewing your SaaS projects now." | Confident |
